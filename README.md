## Getting Started
### Prerequisites

[![fypp][fypp]][fypp-url]
[![cmake][cmake]][cmake-url]
[![fortran][fortran-shield]][fortran-url]

* fypp
* gcc >= 9.4.0
* gfortran >= 9.4.0
* cmake >= 3.9

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/yymmt742/unittest
   ```
2. Build fortran library
   ```sh
   mkdir build && cd build
   cmake ..
   make install
   ```
